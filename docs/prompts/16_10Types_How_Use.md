당신은 20년차 유튜브 관련 신규 아이템을 창출하고 사업성개발 기획을 담당하고 
유튜브고객 시청자들의 심리및 유튜브 개선에 아이디어을 창출해가는 전문가입니다
 최근 다음과 같은 문제가 발생하여 해결할수 있는 방안을 모색 하고자 합니다
[문제점]
  "최근 1개월간 회사 유튜브 채널에 업로드한 신규 영상들의 '평균 시청 지속 시간'이 이전 분기 대비 30% 급감했다."

결과치는  AI 프롬프트 유형에 맞게 제시해 주길 바랍니다

안녕하십니까. 20년 차 유튜브 사업 기획 및 오디언스 심리 분석 전문가입니다.

'평균 시청 지속 시간(AVD)'이 1개월 만에 30% 급감한 것은 채널의 건강 상태에 매우 심각한 **'적신호'**입니다. 이는 단순히 조회수 하락의 문제가 아니라, 시청자가 콘텐츠에 대해 적극적으로 '거부' 신호를 보내고 있다는 의미이며, 유튜브 알고리즘의 추천 시스템에서 즉각적인 페널티를 받게 됩니다.

문제의 원인은 복합적일 수 있으나, 단기간의 급락은 대부분 **'시청자의 기대와 콘텐츠의 실제 가치 간의 불일치'**에서 발생합니다.

이 문제를 해결하기 위해, 귀사의 팀이 즉시 실행하고 데이터를 분석할 수 있는 AI 프롬프트 유형의 진단 및 해결책을 제시합니다.

### 📥 AI 전략 실행 프롬프트: AVD 30% 급락 원인 진단 및 회복 전략

**[프롬프트 유형: 심층 진단 및 개선 전략 (Deep-Dive Diagnosis & Improvement Strategy)]**

**1. 역할 (Persona)**
> 당신은 유튜브 채널 데이터를 분석하여 '평균 시청 지속 시간(AVD)' 하락의 핵심 원인을 진단하고, 시청자 심리를 기반으로 콘텐츠의 매력도를 즉각적으로 회복시킬 전략을 수립하는 '20년 차 유튜브 데이터 전략가'입니다.

**2. 핵심 문제 (Problem Definition)**

| 항목 | 내용 |
| :--- | :--- |
| **현상** | 최근 1개월간 업로드된 신규 영상의 AVD가 이전 분기 대비 30% 급감. |
| **의미** | 콘텐츠가 시청자의 기대치를 충족시키지 못하고 있으며, 이탈률이 심각한 수준임. |

**3. 목표 (Goal)**

| 항목 | 내용 |
| :--- | :--- |
| **핵심 원인 진단** | AVD 급락의 핵심 원인을 '데이터'에 기반하여 3가지 이내로 특정한다. |
| **즉각적 개선** | AVD를 최소 20% 이상 회복시키기 위한 즉각적인 '콘텐츠 개선 액션 플랜'을 도출한다. |

**4. 분석 및 실행 단계 (Analysis & Action Steps)**

**[Step 1: 데이터 심층 분석 (Data Deep-Dive) - '어디서' 이탈하는가?]**

가장 중요한 단계입니다. '감'이 아닌 '데이터'로 원인을 찾아야 합니다.

**1. '시청자 유지율 그래프' 비교 분석**

| 분석 항목 | 세부 내용 |
| :--- | :--- |
| **분석 대상** | [이전 분기 Top 3 영상] vs [최근 1개월 Bottom 3 영상] |
| **핵심 질문 1 (초반)** | **초반 30초 (The Hook):** 그래프가 시작부터 급격히 하락하는가? (초반 이탈) |
| **핵심 질문 2 (중반)** | **특정 이탈 지점 (Dip):** 모든 영상에서 공통적으로 급락하는 특정 구간이 있는가? |
| **핵심 질문 3 (후반)** | **후반부 (Outro):** 이전보다 훨씬 일찍 시청을 중단하는가? |

**2. '유입 경로별 AVD' 분석**

| 분석 항목 | 세부 내용 |
| :--- | :--- |
| **분석 대상** | 유입 경로별 (탐색 기능, 추천 영상, 구독자 등) AVD |
| **핵심 질문** | '탐색 기능'이나 '추천 영상'을 통해 유입된 시청자의 AVD가 '구독자' 유입 AVD보다 현저히 낮은가? |

**[Step 2: 원인 가설 수립 (Hypothesis Generation) - '왜' 이탈하는가?]**

Step 1의 데이터 분석 결과에 따라, 다음과 같은 가설을 세웁니다.

| 가설 | 데이터 근거 (IF) | 가설 내용 (THEN) |
| :--- | :--- | :--- |
| **가설 1 (Hook 실패)** | 초반 30초 이탈률이 급격히 높음 | "신규 영상의 인트로가 시청자 호기심을 자극하지 못하거나 전개가 느리다. 혹은 제목/썸네일과 내용이 불일치하여 '낚시성'으로 인지되었다." |
| **가설 2 (주제 불일치)** | 신규 유입 AVD가 급락 (구독자 AVD는 상대적 유지) | "최근 영상 주제가 기존 채널 정체성이나 핵심 구독자 관심사에서 벗어났다. 알고리즘이 새 시청자에게 노출했지만 그들이 원하는 내용이 아니었다." |
| **가설 3 (포맷 피로도)** | 특정 구간에서 공통적인 이탈(Dip) 발생 | "영상 편집 스타일(템포, 자막, 효과음)이 변경되었거나, 특정 코너/멘트가 시청자에게 지루함이나 거부감을 유발했다." |
| **가설 4 (품질 저하)** | 전반적 AVD 하락 및 댓글 피드백 | "오디오 음질 불량, 화질 저하, 또는 설명의 논리 부족 등 콘텐츠의 '기본 품질'이 저하되어 시청 몰입을 방해했다." |

**[Step 3: 즉각적 개선 조치 (Immediate Action Plan) - '어떻게' 해결할 것인가?]**

가설 검증 및 즉각적인 AVD 회복을 위해 다음 액션을 실행합니다.

| 조치 항목 | 대응 가설 | 세부 액션 플랜 |
| :--- | :--- | :--- |
| **1. 'The Hook' 30초 재설계** | 가설 1 | **'3초 룰' 적용:** 영상 시작 3초 이내에 '끝까지 봐야 하는 이유(핵심 가치)' 제시.<br>**'가치 선(先)제시':** 본론의 가장 매력적인 결과나 하이라이트를 인트로에 전진 배치. |
| **2. 데이터 기반 편집** | 가설 3 | 시청자 유지율 그래프의 이탈(Dip) 구간을 1초 단위로 분석.<br>해당 구간 속도를 1.2배 높이거나, 불필요한 멘트 삭제, 시각적 효과(자막, 줌인) 추가. |
| **3. A/B 테스트** | 가설 1, 2 | 동일 주제로 인트로/포맷이 다른 2가지 버전 제작 테스트.<br>썸네일과 제목을 2~3종으로 변경하며 AVD 변화를 실시간 추적. |
| **4. 시청자 직접 소통** | 가설 2, 4 | 커뮤니티 탭이나 댓글 고정을 통해 "최근 영상 중 가장 아쉬웠던 점?", "다음 주제" 등 구체적인 피드백 요청. |

**[Step 4: 중장기 전략 (Long-Term Strategy) - '지속' 가능한가?]**

| 전략 | 세부 내용 |
| :--- | :--- |
| **1. 핵심 시청자 페르소나 재정의** | 우리의 콘텐츠를 끝까지 보는 사람들은 누구인가? 그들의 '진짜 문제'는 무엇인가? |
| **2. 콘텐츠 포트폴리오 재구성** | 기존 성공 공식(AVD가 높았던) 70% + 새로운 실험(신규 포맷/주제) 30% 비율을 유지하며 안정성과 확장성을 동시 확보. |

**5. 기대 결과 (Expected Outcome)**

| 항목 | 기대 결과 |
| :--- | :--- |
| **원인 진단** | 1주일 이내 AVD 급락의 핵심 원인 1~2가지 특정 |
| **단기 성과** | 개선안이 적용된 신규 영상의 AVD가 이전 영상 대비 즉각적인 10% 상승 |
| **중기 성과** | 1개월 이내 채널 전체 AVD를 이전 분기 대비 80% 수준까지 회복 |
| **부가 효과** | 시청자 피드백을 반영한 '소통하는 채널' 이미지 구축 및 충성도 상승 |

### 👨‍💼 전문가 코멘트

AVD의 30% 급락은 매우 고통스러운 지표이지만, 동시에 **"시청자가 무엇을 원하지 않는지"**를 가장 명확하게 알려주는 데이터입니다.

시청자가 영상을 클릭한 순간, 그들은 썸네일과 제목을 통해 '특정 가치'를 기대합니다. AVD 급락은 그 '약속'이 영상 초반에 깨졌다는 뜻입니다. 지금 당장 최근 영상들의 '시청자 유지율 그래프'부터 확인하시고, 30초의 '골든 타임'을 사수하기 위한 인트로 전략을 전면 재검토하시길 강력히 권고합니다.

혹시 분석 과정에서 특정 데이터(예: 시청자 유지율 그래프의 패턴)를 공유해주시면, 더 구체적인 원인 진단과 다음 스텝을 제안해 드릴 수 있습니다.
